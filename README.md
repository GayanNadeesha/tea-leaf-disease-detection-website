# Tea Leaf Disease Detection System 🌿

# Overview

A web-based AI application that detects diseases in tea leaves using Convolutional Neural Networks (CNNs). Farmers can upload leaf images and get disease identification, causes, treatment methods, and recommended medicine sources to make better crop management decisions.

---

## Features

- Upload tea leaf images through a simple web interface  
- Detect tea leaf diseases automatically using CNN  
- Provides disease name, causes, treatments, and recommended medicine sources  
- Helps farmers make informed decisions to manage crops effectively

## Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Python, Flask  
- Machine Learning: CNN (Convolutional Neural Network)  
- Other Tools: OpenCV, NumPy, Pandas  

---

## How to Run

1. Clone this repository:  
   `git clone <your-repo-link>`  
2. Navigate to project folder:  
   `cd tea-leaf-disease-detection-website`  
3. Install dependencies:  
   `pip install -r requirements.txt`  
4. Run the Flask app:  
   `python app.py`  
5. Open your browser and go to `http://127.0.0.1:5000`
